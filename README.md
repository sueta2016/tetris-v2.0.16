# Tetris

## Setup and Testing

Follow these simple steps to set up the environment and run tests:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/sueta2016/tetris-v2.0.16.git
   ```

2. **Navigate to Project Directory:**
   ```bash
   cd tetris-v2.0.16
   ```

3. **Install Dependencies:**
   ```bash
   cargo build
   ```

4. **Run Tests:**
   ```bash
   cargo test
   ```
